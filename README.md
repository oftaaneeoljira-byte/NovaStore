# NovaStore – Modern E-Commerce UI

NovaStore is an e-commerce frontend built using HTML, CSS, and JavaScript.

## 🚀 Features

- Multi-page responsive layout
- Modern glassmorphism navbar
- Product grid using CSS Grid
- Login & Register system
- Clean folder structure
- Ready for backend integration

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

## 📦 Project Structure

novastore/
│
├── index.html
├── css/style.css
├── script/auth.js
└── README.md

## 🌍 Deployment Guide

### Option 1: GitHub Pages

1. Push project to GitHub
2. Go to Repository Settings
3. Click Pages
4. Select branch (main)
5. Save
6. Your site will be live

### Option 2: Netlify

1. Go to https://netlify.com
2. Drag and drop the project folder
3. Done – instant live link

### Option 3: Vercel

1. Install Vercel CLI or use website
2. Import GitHub repo
3. Deploy
